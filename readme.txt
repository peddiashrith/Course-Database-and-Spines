The base file is home.html. 

No installation required.

All files are HTML, with JavaScript and CSS included. Hence your browser must support JavaScript and CSS. There is a small application of Bootstrap, which is accessed online. Ensure that your computer is connected to the Internet, as some fonts are also accessed from Google.

The credits displayed in the modal boxes of the courses are in the form L-T-P-C: Lectures, Tutorials, Practicals, Credits.

If an arrow points from node A to node B, then A is prerequisite of B.
